Angular

I was involved in setting up angular environment for our project. I invented structure of project and created application routing. I have created promises for loading animal views through reading QR code and also promises for interaction with our web server.

I also enforced best practises for writing all our controllers/directives/services. That means that we are using IIFE(immediately invoked function expressions) and that every controller/directive/filter is in separate file. No more than one controller/directive/filter per file !

Gulp

I have installed gulp in order to have option to easily compile all javascript files into one file. That is standard between javascript aplications and increases developer efficiency due to automation of tasks.

Golang

I created simple web server which supports cross domain requests which are required by our mobile app to rate animals. Web server is written in golang and runs on openshift platform. It is easily maintanable and scalable written server so that means that adding additional functionality(e.g. comments) would be easy and fast.

MongoDB

For our database we chose NO-SQL based MongoDB. I created cartridge on openshift platform and integrated with our golang web server.
